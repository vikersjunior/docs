The Workout Library is your starting point for every session. It holds six beginner-friendly routines covering mobility, endurance, strength, and weight loss — and three filter controls let you zero in on exactly what you need without scrolling through options that don't fit. Filters apply the moment you interact with them, so there's no submit button to click; just adjust a dropdown or start typing and the list updates instantly.

## Opening the Workout Library

Navigate to the Workout Library page in your browser. You'll see all six routines displayed as cards, along with the filter bar at the top of the page.

## Using the Filters

### Goal Filter

The **Goal** dropdown narrows the list by the primary fitness objective of each routine.

| Option | What it shows |
|---|---|
| All Goals | Every routine in the library |
| Weight Loss | Routines designed to burn calories and reduce body fat |
| Strength | Routines focused on building muscle and functional power |
| Mobility | Routines that improve flexibility, range of motion, and recovery |
| Endurance | Routines that build cardiovascular stamina over time |

### Level Filter

The **Level** dropdown filters by the physical demand and experience expected.

| Option | What it shows |
|---|---|
| All Levels | Every routine regardless of difficulty |
| Beginner | Designed for people new to exercise — no prior fitness background required |
| Easy | Low-intensity sessions suitable for active recovery or lighter effort days |
| Intermediate | A step up in complexity or intensity; some fitness base is helpful |

### Keyword Search

The **Keyword Search** field matches your text against each routine's name, description, and individual exercise list in real time. This is especially useful when you already know an exercise you want to include.

<Tip>
  Type a single muscle group or movement — like "plank" or "squat" — to surface every routine that contains that exercise, even if the routine name doesn't mention it.
</Tip>

## Combining Filters for Faster Results

All three controls work together simultaneously. You can set a goal, pick a level, and type a keyword at the same time — the library shows only the routines that satisfy every active filter.

<Steps>
  <Step title="Set your goal">
    Open the **Goal** dropdown and select the outcome you're training toward — for example, **Strength**.
  </Step>
  <Step title="Pick your level">
    Open the **Level** dropdown and choose how demanding the session should be — for example, **Beginner**.
  </Step>
  <Step title="Refine with a keyword (optional)">
    Type a word into the **Keyword Search** field to narrow results further. The library updates as you type — no need to press Enter.
  </Step>
  <Step title="Browse your results">
    Review the matching routine cards. Each card shows the routine name, duration, goal, level, required equipment, and exercise list.
  </Step>
</Steps>

**Example:** To find a strength workout for beginners, set **Goal** to **Strength** and **Level** to **Beginner**. The library will show _Core Strength Builder_ (15 min, mat) and _Starter Strength Session_ (22 min, light dumbbells).

<Note>
  Filters apply instantly — there is no submit or search button. If no routines appear, check that your keyword doesn't conflict with your dropdown selections, then try clearing one filter at a time.
</Note>

## Resetting the Filters

To return to the full library, set both dropdowns back to **All Goals** and **All Levels**, and clear any text from the keyword field. All six routines will reappear immediately.
